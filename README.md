# Feradi WPI - Forum Era Adil Warung Paralegal Indonesia

Selamat datang di repositori website resmi **Feradi WPI (Forum Era Adil Warung Paralegal Indonesia)** & **Subur Jaya Law Firm**.  

Website ini dirancang untuk menampilkan profil institusi, area praktik, layanan bantuan hukum, serta memberikan akses mudah bagi calon klien untuk berkonsultasi.

---

## Status Proyek Saat Ini

> **Perhatian:**  
> Untuk saat ini, versi website ini masih dalam bentuk **HTML Statis murni** (menggunakan Tailwind CSS via CDN dan JavaScript murni untuk interaktivitas).

Proyek ini sedang dalam tahap pembangunan aktif untuk dilanjutkan ke tahap pengembangan berikutnya.

### Rencana Pengembangan

Rencana pengembangan ke depan mungkin mencakup:

- Integrasi dengan framework modern (seperti React, Next.js, atau Laravel)
- Pembuatan backend untuk memproses formulir kontak/konsultasi
- Integrasi CMS (Content Management System) untuk pembaruan berita/artikel secara dinamis
- Optimasi aset dan SEO tingkat lanjut

---

## Fitur & Seksi yang Tersedia (Versi HTML)

- **Hero Section**  
  Desain modern dengan latar belakang abstrak dan Call-to-Action (CTA).

- **Statistik**  
  Menampilkan data rasio kemenangan, jumlah kantor, dan pencapaian lainnya.

- **Profil Institusi (Tentang Kami)**  
  Penjelasan mengenai sinergi Feradi WPI dan Subur Jaya Law Firm beserta profil pendiri.

- **Area Praktik & Layanan**  
  Rincian spesialisasi hukum dan ruang lingkup bantuan hukum.

- **Metodologi (Proses Kerja)**  
  Alur 4 langkah dari evaluasi hingga resolusi tuntas.

- **Nilai Inti (Keunggulan)**  
  Empat pilar kepercayaan firma.

- **Testimoni Klien**  
  Bukti dedikasi dalam bentuk ulasan klien.

- **FAQ (Pertanyaan Umum)**  
  Akordion interaktif untuk menjawab pertanyaan seputar layanan.

- **Formulir Konsultasi & Kontak**  
  Formulir interaktif yang otomatis mengarahkan klien ke WhatsApp admin.

- **Modals**  
  Pop-up untuk:
  - Ketentuan Layanan  
  - Kebijakan Privasi  

---

## Teknologi yang Digunakan

Pada tahap awal ini, website dibangun menggunakan:

- **HTML5 & CSS3**
- **Tailwind CSS** (via CDN untuk rapid prototyping)
- **Vanilla JavaScript**  
  (untuk toggle menu mobile, efek scroll navbar, FAQ, dan modals)
- **Google Fonts (Inter)**
- **FontAwesome** (untuk ikon SVG)
